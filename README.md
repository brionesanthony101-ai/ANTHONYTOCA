# ANTHONYTOCA
# 📊 Análisis Competitivo de Precios

Dashboard desarrollado en Power BI para el monitoreo del Precio de Venta 
al Consumidor (PVC), comportamiento competitivo y participación por categoría 
en el mercado de productos de limpieza e higiene.

---

## 🎯 Objetivo

Monitorear la evolución de precios por fabricante, identificar brechas 
competitivas y analizar la distribución de productos por categoría 
para apoyar decisiones comerciales estratégicas.

---

## 📌 ¿Qué muestra el dashboard?

- **KPI Total PDV** – Recuento de puntos de venta analizados
- **Evolución del PVC por Fabricante** – Comparativo mensual entre 
  competidores: Jabonería Wilson, Unilever, Colgate, La Fabril, 
  Alicorp, entre otros
- **Precio Oferta y Porcentualidad** – Tabla comparativa PVC vs PVO 
  con % de descuento por SKU
- **Distribución por Categoría** – Participación de Líquido, Polvo, 
  Crema, Barra y Otros

---

## 🔍 Filtros disponibles

| Filtro | Descripción |
|---|---|
| Año / Mes | Período de análisis |
| Cadena | Canal de distribución |
| Categoría | Tipo de producto |
| Sub-categoría | Segmento específico |
| Gramaje BI | Tamaño del producto |

---

## 🛠️ Herramientas utilizadas

- Power BI Desktop
- Power Query (ETL y transformación de datos)
- DAX (medidas y KPIs)
- Excel (fuente de datos)
